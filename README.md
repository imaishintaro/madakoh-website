# マダミス広報部（マダこー）公式ウェブサイト

マダミス広報部（通称：マダこー）は、マーダーミステリー（略してマダミス）を広めるための活動を行うレビュー集団です。このリポジトリは公式ウェブサイトのソースコードを管理しています。

## サイト概要

マダミス広報部の活動やメンバー紹介、イベント情報などを掲載しています。
